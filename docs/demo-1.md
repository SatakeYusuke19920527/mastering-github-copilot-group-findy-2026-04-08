# DEMO 1: 欠席者AIによる振替自動振り分け

## 概要

欠席した生徒に対して、AIが最適な振替授業を自動提案・実行するフローをデモする。
Copilot CLIの `/plan` → 実装 → `/fleet` → `/review` → `/diff` の一気通貫ワークフローを見せる。

---

## DEMOフロー

### Step 1: `/plan` — 実装計画の立案

```
/plan 欠席管理の欠席一覧にある生徒を、時間割で空いている教室＋同じ学年の授業＋その生徒が通常参加していない授業に振替する機能を実装したい。

要件:
1. 欠席一覧画面に「AI振替提案」ボタンを追加
2. AIが以下の条件でマッチング:
   - 空き教室がある時間帯
   - 同じ学年の授業
   - その生徒が通常参加していない授業（重複回避）
3. カード状態遷移: 欠席一覧 → 振替実施予定 → 振替完了
4. 時間割の受講生徒一覧に「名前(振替)」の形式で振替メンバーを表示
```

**見せどころ**: Copilotがコードベースを分析し、構造化された計画を立てる様子

---

### Step 2: 計画の確認・調整

- `plan.md` が生成される
- 必要に応じて計画を修正
- 実装開始を指示:

```
plan.mdの計画に従って実装を開始して
```

**見せどころ**: 人間が計画をレビュー・承認してから実装に進むフロー

---

### Step 3: `/fleet` — 並列実装

```
/fleet plan.mdに従って、以下を並列で実装して:
1. 振替ロジック（src/lib/ai/reschedule.ts の拡張）
2. 欠席一覧UIの状態遷移（欠席一覧→振替実施予定→振替完了）
3. 時間割への振替メンバー表示
4. APIルートの追加
```

**見せどころ**: 複数ファイルを並列サブエージェントで同時に実装する高速開発

---

### Step 4: 動作確認

```
npm run build でビルドが通るか確認して
```

**見せどころ**: Copilotが自動でビルドエラーを検知・修正

---

### Step 5: `/review` — コードレビュー

```
/review
```

**見せどころ**: AIが自動でコード品質・セキュリティ・バグをチェック

---

### Step 6: `/diff` — 変更差分の確認

```
/diff
```

**見せどころ**: 全変更の差分を一覧で確認

## 期待される成果物

### データモデル（absences コンテナ）

```typescript
interface Absence {
  id: string;
  studentId: string;
  studentName: string;
  gradeLevel: string;
  originalScheduleId: string;
  date: string;
  status: "absent" | "reschedule_pending" | "reschedule_done";
  rescheduleInfo?: {
    targetScheduleId: string;
    targetDate: string;
    targetTimeSlot: string;
    suggestedBy: "ai";
  };
}
```

### UI状態遷移

```
┌─────────────┐     AI振替提案      ┌──────────────────┐     振替実施      ┌─────────────┐
│  欠席一覧   │ ─────────────────→ │  振替実施予定     │ ──────────────→ │  振替完了    │
│  (absent)   │                     │(reschedule_pending)│                  │(reschedule_done)│
└─────────────┘                     └──────────────────┘                  └─────────────┘
```

### 時間割での表示

```
月曜日 17:00-18:00 数学（中2）
├── 田中太郎
├── 佐藤花子
└── 山田一郎(振替)  ← 振替メンバー
```

## 使用するCopilot CLI機能

| 機能 | 用途 |
|------|------|
| `/plan` | 実装計画の自動生成 |
| `Shift+Tab` | plan → interactive モード切替 |
| `/fleet` | 並列サブエージェントで高速実装 |
| `/review` | AIコードレビュー |
| `/diff` | 変更差分の確認 |
| `@ファイル名` | 特定ファイルのコンテキスト参照 |
